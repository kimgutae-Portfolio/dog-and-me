# きみとの映画 — First MVP

愛犬の写真とエピソードから、実写映画のような思い出映像を注文できるサービスの1次開発版です。

このリポジトリでは、最初に検証すべき「サービスを理解する → 思い出を入力する → 相談を受け付ける → 制作状況を見る」という導線を実装しています。AI動画生成、決済、大容量アップロードはまだ自動化せず、品質と注文体験を先に検証する方針です。

## 1次版でできること

- 日本語のブランドランディングページ
- 料金・制作フロー・FAQの確認
- 5ステップのストーリー入力
- 端末内への入力内容の自動保存
- 写真選択とファイル名の確認（実送信は未接続）
- 相談受付と制作室デモ
- レスポンシブ・キーボード操作・reduce motion対応
- ヘルスチェック API (`/api/health`)

## 起動

Node.js 22.13 以上を推奨します。

```bash
npm ci
npm run dev
```

本番ビルドとテスト:

```bash
npm run build
npm test
```

## 主要ページ

- `/` — ランディング
- `/story` — 思い出入力ウィザード
- `/studio` — 制作室デモ
- `/api/health` — 稼働確認

## データについて

1次版の入力データはブラウザの `localStorage` にのみ保存されます。写真本体は外部へ送信しません。写真のアップロード、認証、注文の永続化はSupabase接続後に有効化します。
